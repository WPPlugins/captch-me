== Installation ==

Ci-dessous les étapes nécessaires à l'installation du plugin Wordpress Captch Me:

1. S'inscire à Attentive Ads en tant qu'éditeur, en choisissant le format publicitaire Captch Me : [Inscription](http://www.attentiveads.com/inscription-monetisation.html)
2. Installer le plugin depuis Wordpress ou copier le répertoire captchme dans le répertoire /wp-content/plugins/.
3. Activer le plugin via le menu 'Extensions Installées' de wordpress
4. Renseigner les clefs publique, privée et d'authentification dans la page de réglage du plugin.
5. Activer/Désactiver les captcha Captch Me sur la page d'inscription et dans les commentaires.
6. Ajouter Captch Me dans vos formulaires "Contact Form 7"
7. Activer/Désactiver Show Me et Leave Me
8. Ajouter Access Me sur vos pages pour protéger vos liens et le contenu des pages (par exemple un bouton "Lire la suite" avec un captcha)
